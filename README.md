# TradeMind AI

Asistente de trading inteligente con análisis multi-temporalidad.

## Características

- **5 Pestañas**: Dashboard, Gráficos, Operación, Análisis IA, Noticias
- **22 Activos**: 20 pares Forex + 2 criptomonedas
- **3 Temporalidades**: 1D (Dirección), 1H (Estrategia), 5M (Ejecución)
- **EMA 50**: Indicador en todas las temporalidades
- **Estrategias**: New York (Impulse + Continuation) y Smart Money (Manipulation)
- **Datos en tiempo real**: Precios actualizados
- **PWA**: Instalable como aplicación nativa

## Instalación

```bash
npm install
npm run build
npm start
```

## Tecnologías

- Next.js 14
- TypeScript
- Tailwind CSS 3.4
- shadcn/ui
- TradingView Widgets
